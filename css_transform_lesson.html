🔄 CSS Transform

Элементтерді түрлендіру мен анимациялау

🎯 Transform дегеніміз не?

CSS Transform элементтерді олардың орналасқан орнын өзгертпей-ақ жылжыту, бұру, масштабтау және қию мүмкіндігін береді.

Мені басыңыз!
🎨 Артықшылықтары:
  • Жылдам орындалады (GPU арқылы)
  • Басқа элементтерге әсер етпейді
  • Анимацияға өте жақсы
  • 3D эффекттер жасауға болады
⚠️ Ескертулер:
  • Document flow-ды өзгертпейді
  • Z-index мәселелері болуы мүмкін
  • Мәтін blur болуы мүмкін
  • Кейбір браузерлердегі қолдау

🔧 Негізгі Transform функциялары

📍 Translate - жылжыту

Элементті X және Y осьтері бойынша жылжытады

/* X осі бойынша */ transform: translateX(50px); /* Y осі бойынша */ transform: translateY(-30px); /* Екеуін бірдей */ transform: translate(50px, -30px); /* 3D жылжыту */ transform: translate3d(50px, -30px, 20px);

Қасиеттері:

translateX() translateY() translateZ() translate() translate3d()

🔄 Rotate - бұру

Элементті белгілі бір бұрышқа бұрады

/* Z осі бойынша бұру */ transform: rotate(45deg); /* X осі бойынша бұру */ transform: rotateX(60deg); /* Y осі бойынша бұру */ transform: rotateY(90deg); /* 3D бұру */ transform: rotate3d(1, 1, 0, 45deg);

Қасиеттері:

rotate() rotateX() rotateY() rotateZ() rotate3d()

📏 Scale - масштабтау

Элементтің өлшемін үлкейтеді немесе кішірейтеді

/* Барлық жақты */ transform: scale(1.5); /* X осі бойынша */ transform: scaleX(2); /* Y осі бойынша */ transform: scaleY(0.5); /* X және Y бөлек */ transform: scale(1.2, 0.8); /* 3D масштабтау */ transform: scale3d(1.2, 0.8, 1.5);

Қасиеттері:

scale() scaleX() scaleY() scaleZ() scale3d()

🔀 Skew - қию

Элементті X және Y осьтері бойынша қияды

/* X осі бойынша қию */ transform: skewX(15deg); /* Y осі бойынша қию */ transform: skewY(10deg); /* Екеуін бірдей */ transform: skew(15deg, 10deg); /* Тек X осі */ transform: skew(20deg);

Қасиеттері:

skew() skewX() skewY()

🔢 Matrix - матрица

Барлық түрлендірулерді бір функцияда біріктіреді

/* 2D матрица */ transform: matrix(a, b, c, d, tx, ty); /* 3D матрица */ transform: matrix3d( a1, b1, c1, d1, a2, b2, c2, d2, a3, b3, c3, d3, a4, b4, c4, d4 ); /* Мысал */ transform: matrix(1, 0.2, 0.3, 1, 20, 10);

Параметрлер:

  • a, d: масштабтау
  • b, c: қию
  • tx, ty: жылжыту

🎬 Бірнеше функция

Бір элементте бірнеше түрлендіруді қолдану

/* Реттілік маңызды! */ transform: translate(50px, 20px) rotate(45deg) scale(1.2); /* 3D комбинация */ transform: perspective(500px) rotateX(30deg) rotateY(45deg) translateZ(50px);
⚠️ Маңызды! Transform функцияларының реттілігі маңызды. Олар солдан оңға қарай қолданылады.

📍 Transform Origin

transform-origin қасиеті түрлендіру орталығын анықтайды. Әдепкі мәні элементтің ортасы (50% 50%).

🎯 Орталықтан бұру

.element { transform-origin: center; /* немесе */ transform-origin: 50% 50%; }

📍 Сол жақ жоғарғы бұрыштан

.element { transform-origin: top left; /* немесе */ transform-origin: 0% 0%; }

📍 Оң жақ төменгі бұрыштан

.element { transform-origin: bottom right; /* немесе */ transform-origin: 100% 100%; }

📏 Нақты координаттар

.element { transform-origin: 20px 30px; /* немесе */ transform-origin: 25% 75%; }
/* Мүмкін мәндер */ transform-origin: center; /* 50% 50% */ transform-origin: top; /* 50% 0% */ transform-origin: bottom; /* 50% 100% */ transform-origin: left; /* 0% 50% */ transform-origin: right; /* 100% 50% */ transform-origin: top left; /* 0% 0% */ transform-origin: top right; /* 100% 0% */ transform-origin: bottom left; /* 0% 100% */ transform-origin: bottom right; /* 100% 100% */ /* Нақты мәндер */ transform-origin: 20px 30px; /* X: 20px, Y: 30px */ transform-origin: 25% 75%; /* X: 25%, Y: 75% */ transform-origin: 50px 25% 10px; /* X: 50px, Y: 25%, Z: 10px (3D) */

🌐 3D Transform

👁️ Perspective

3D эффект үшін перспектива орнатады

/* Ата-ана элементте */ .container { perspective: 500px; } /* Немесе элементте */ .element { transform: perspective(500px) rotateY(45deg); }

Perspective мәндері:

  • 100px-300px: күшті 3D эффект
  • 500px-800px: орташа эффект
  • 1000px+: нәзік эффект

🔄 3D бұру

X, Y, Z осьтері бойынша бұру

/* X осі бойынша (жоғары-төмен) */ transform: rotateX(45deg); /* Y осі бойынша (сол-оң) */ transform: rotateY(45deg); /* Z осі бойынша (сағат тілі) */ transform: rotateZ(45deg); /* Барлығын бірдей */ transform: rotate3d(1, 1, 1, 45deg);

📦 Transform Style

3D балалар элементтері үшін

.parent { transform-style: preserve-3d; /* Балалар элементтерінің 3D позициясын сақтайды */ } .child { transform: translateZ(50px); }

⚡ Өнімділік және оңтайландыру

🚀 GPU үдету

Бұл қасиеттер GPU арқылы үдетіледі және өте жылдам жұмыс істейді:

✅ Жылдам:
  • transform
  • opacity
  • filter
⚠️ Баяу:
  • width/height
  • top/left
  • margin/padding

❌ Жақсы емес

/* Layout-ты қайта есептейді */ .element { left: 100px; top: 50px; width: 200px; height: 150px; } /* Анимация үшін өте баяу */ @keyframes badMove { from { left: 0; } to { left: 100px; } }

✅ Жақсы

/* Layout-ты өзгертпейді */ .element { transform: translate(100px, 50px) scale(1.2); } /* Анимация үшін өте жылдам */ @keyframes goodMove { from { transform: translateX(0); } to { transform: translateX(100px); } }

🎯 Маңызды кеңестер

  • will-change қасиетін пайдаланыңыз анимация алдында
  • transform3d() немесе translateZ(0) GPU үдетуін мәжбүрлеу үшін
  • Анимациядан кейін will-change: auto орнатыңыз
  • Көп элементке бір уақытта анимация жасамаңыз

🌐 Браузер қолдауы

✅ Толық қолдау

  • Chrome 36+
  • Firefox 16+
  • Safari 9+
  • Edge барлық нұсқалары
  • IE 10+ (prefix керек)

⚠️ Prefix керек

/* Ескі браузерлер үшін */ .element { -webkit-transform: rotate(45deg); -moz-transform: rotate(45deg); -ms-transform: rotate(45deg); -o-transform: rotate(45deg); transform: rotate(45deg); }

🔍 Тексеру

/* CSS Feature Query */ @supports (transform: rotate(45deg)) { .element { transform: rotate(45deg); } } /* JavaScript тексеру */ if ('transform' in document.body.style) { // Transform қолдау бар }

🛠️ Fallback

/* Fallback мысалы */ .element { /* Ескі браузерлер үшін */ position: relative; left: 50px; /* Жаңа браузерлер үшін */ transform: translateX(50px); left: auto; }

💡 Пайдалы кеңестер

✅ Жақсы практикалар:

  • GPU үдету үшін transform қолданыңыз
  • Анимация үшін transition пайдаланыңыз
  • Transform-origin дұрыс орнатыңыз
  • 3D эффекттер үшін perspective қосыңыз
  • Will-change қасиетін пайдаланыңыз
  • Префикстерді ұмытпаңыз (ескі браузерлер)

❌ Тыйым салынған:

  • Layout өзгерту үшін transform қолдану
  • Көп элементке бір уақытта анимация
  • Ұзын анимациялар жасау
  • Z-index мәселелерін елемеу
  • Accessibility ескермеу
  • Мәтін blur болуын елемеу

🎯 Негізгі ережелер:

🚀 Өнімділік:
  • Transform + opacity жылдам
  • Width/height/position баяу
  • GPU layer құру керек
🎨 Дизайн:
  • Нәзік анимациялар жақсы
  • Easing функцияларын қолданыңыз
  • Контекстке сай таңдаңыз
🛠️ Техникалық:
  • Браузер қолдауын тексеріңіз
  • Fallback опцияларын дайындаңыз
  • Performance monitoring жасаңыз
