Интерактивті сабақ - пайдаланушыдан мәліметтер алу

HTML Forms туралы

HTML Forms - веб-беттерде пайдаланушылардан мәліметтер жинау үшін қолданылатын элементтер жиынтығы. Форма арқылы мәтін, сандар, файлдар және басқа деректерді жібереуге болады.

Формалардың маңызды элементтері:

  • <form> - форманың негізгі контейнері
  • <input> - мәліметтер енгізу өрісі
  • <textarea> - көп жолды мәтін өрісі
  • <select> - тізімнен таңдау
  • <button> - форманы жіберу батырмасы
  • <label> - өрістердің атауы

Қарапайым форма мысалы:

Form элементінің негіздері

<form> тегі барлық форма элементтерін орайды. Ол екі маңызды атрибутқа ие:

  • action - деректер қайда жіберілетінін көрсетеді
  • method - деректер қалай жіберілетінін көрсетеді (GET немесе POST)
<!-- Негізгі форма құрылымы --> <form action="/submit" method="POST"> <!-- Форма элементтері осында болады --> <label for="username">Пайдаланушы аты:</label> <input type="text" id="username" name="username"> <button type="submit">Жіберу</button> </form>

Method айырмашылығы:

GET Method

Ерекшеліктері:

  • URL-да көрінеді
  • Қауіпсіз емес
  • Іздеу үшін жақсы
  • Кіші деректер үшін
example.com/search?q=HTML+forms

POST Method

Ерекшеліктері:

  • URL-да көрінбейді
  • Қауіпсіз
  • Деректерді жіберу үшін
  • Үлкен деректер үшін
Деректер HTTP body-да жіберіледі

Input элементінің түрлері

type атрибуты арқылы input элементінің түрін анықтайды. Әртүрлі типтер әртүрлі мүмкіндіктер береді.

text - Қарапайым мәтін

<input type="text" placeholder="Мәтін">

email - Электрондық пошта

<input type="email" placeholder="email">

password - Құпия сөз

<input type="password">

number - Сан

<input type="number" min="1" max="100">

date - Күн

<input type="date">

time - Уақыт

<input type="time">

tel - Телефон

<input type="tel">

url - Веб-сайт

<input type="url">

color - Түс таңдау

<input type="color">

range - Диапазон

<input type="range" min="0" max="100">

file - Файл таңдау

<input type="file">

search - Іздеу

<input type="search">

Басқа форма элементтері

Radio Button - Бір таңдау

<input type="radio" name="size" value="m"> <label>Орташа</label>

Checkbox - Көп таңдау

<input type="checkbox" value="music"> <label>Музыка</label>

Select - Тізімнен таңдау

<select> <option value="almaty">Алматы</option> </select>

Textarea - Көп жолды мәтін

<textarea rows="4" cols="50"> </textarea>

Button түрлері

<button type="submit">Жіберу</button> <button type="reset">Тазалау</button>

Fieldset - Топтастыру

Жеке мәліметтер
<fieldset> <legend>Жеке мәліметтер</legend> <!-- Өрістер --> </fieldset>

Форма валидациясы

HTML-да кіру деректерін тексеру үшін арнайы атрибуттар бар. Бұл пайдаланушыға дұрыс деректер енгізуге көмектеседі.

Валидация атрибуттары:

Бұл өрісті толтыру қажет
Дұрыс!
Кемінде 3 символ енгізіңіз
Дұрыс!
Тек сандар енгізуге болады
Дұрыс!
1 мен 100 арасындағы сан енгізіңіз
Дұрыс!
<!-- Валидация атрибуттары --> <input type="text" required minlength="3"> <input type="number" min="1" max="100"> <input type="email" pattern="[a-z0-9._%+-]+@[a-z0-9.-]+\.[a-z]{2,}$">

💡 Валидация кеңестері:

  • required - өрісті міндетті етеді
  • minlength/maxlength - мәтін ұзындығын шектейді
  • min/max - сандық мәндерді шектейді
  • pattern - regex үлгісі арқылы тексереді
  • type="email" - email форматын тексереді

Білімді тексеру

1. Форма деректерін жіберу үшін қай method қауіпсіз?
  • GET
  • POST
  • PUT
  • DELETE
2. Қай input type автоматты түрде email форматын тексереді?
  • text
  • url
  • email
  • tel
3. Radio button-дар арасында бір ғана таңдауға мүмкіндік беру үшін не істеу керек?
  • Бірдей id беру
  • Бірдей name беру
  • Бірдей class беру
  • Бірдей value беру
4. Өрісті міндетті ету үшін қай атрибут қолданылады?
  • required
  • mandatory
  • necessary
  • important
5. <label> элементі неге қолданылады?
  • Форманы безендіру үшін
  • Валидация үшін
  • Input өрістеріне атау беру үшін
  • Форманы жіберу үшін
6. Көп жолды мәтін енгізу үшін қай элемент қолданылады?
  • <input type="text">
  • <input type="longtext">
  • <textbox>
  • <textarea>
