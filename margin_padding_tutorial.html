🏗️ CSS Margin & Padding

Элементтер арасындағы аралық пен ішкі кеңістікті басқару

📚 Кіріспе

Margin және Padding - веб-дизайндағы ең маңызды аралық құралдары. Дұрыс қолданылған spacing дизайнды таза, оқуға ыңғайлы және кәсіби етеді.

🎯 Осы сабақта үйренетініміз:

  • Margin vs Padding айырмашылығы
  • Box Model және оның құрылысы
  • Shorthand нотациялары
  • Auto margin және центрлеу
  • Negative margin техникалары
  • Margin collapsing ережелері
  • Responsive spacing стратегиялары
  • CSS Grid және Flexbox-тағы spacing

🔍 Box Model визуализациясы

CONTENT

Margin: Элементтің сыртқы жағындағы аралық
Padding: Элементтің ішкі жағындағы аралық

📏 Margin негіздері

🔴 Margin қасиеттері

Сыртқы аралық: Элементтердің арасындағы қашықтық

/* Жеке жақтардың margin-дері */ .margin-top { margin-top: 20px; } .margin-right { margin-right: 20px; } .margin-bottom { margin-bottom: 20px; } .margin-left { margin-left: 20px; } /* Қысқа жазу - барлық жақтар */ .margin-all { margin: 20px; } /* Қысқа жазу - 2 мән (vertical horizontal) */ .margin-vh { margin: 20px 10px; } /* Қысқа жазу - 4 мән (top right bottom left) */ .margin-full { margin: 10px 15px 20px 5px; } /* Пиксель емес өлшемдер */ .margin-em { margin: 1em; } .margin-rem { margin: 1.5rem; } .margin-percent { margin: 5%; } /* Автоматты margin */ .margin-auto { margin: 0 auto; } /* Горизонталь центр */ /* Negative margin */ .margin-negative { margin: -10px; } /* Жекелеген жағына negative */ .margin-negative-top { margin-top: -15px; }

🎯 Auto margin және центрлеу

Автоматты орналасу: Элементтерді центрлеу және орналастыру

/* Горизонталь центрлеу */ .center-horizontal { margin: 0 auto; width: 300px; /* Міндетті: анықталған ені */ } /* Тек оң жаққа итеру */ .push-right { margin-left: auto; } /* Тек сол жаққа итеру */ .push-left { margin-right: auto; } /* Flexbox-тағы auto margin */ .flex-container { display: flex; } .flex-item-right { margin-left: auto; /* Оң жаққа итеру */ } .flex-item-center { margin: 0 auto; /* Центрге орналастыру */ } /* Grid-тағы auto margin */ .grid-container { display: grid; grid-template-columns: 1fr 200px 1fr; } .grid-center { margin: 0 auto; grid-column: 2; } /* Вертикалдық центрлеү (сирек) */ .vertical-center { margin-top: auto; margin-bottom: auto; height: 100px; /* Анықталған биіктік */ } /* Толық центрлеу */ .full-center { margin: auto; position: absolute; top: 0; right: 0; bottom: 0; left: 0; width: 200px; height: 100px; }

💡 Auto margin ережелері:

  • Horizontal auto: Элементтің ені анықталған болуы керек
  • margin: 0 auto: Горизонталь центрлеу
  • margin-left: auto: Оң жаққа итеру
  • Flexbox-та: Auto margin әртүрлі жұмыс істейді
  • Inline элементтер: Auto margin жұмыс істемейді

📦 Padding негіздері

🔵 Padding қасиеттері

Ішкі аралық: Контент пен border арасындағы кеңістік

/* Жеке жақтардың padding-дері */ .padding-top { padding-top: 20px; } .padding-right { padding-right: 20px; } .padding-bottom { padding-bottom: 20px; } .padding-left { padding-left: 20px; } /* Қысқа жазу - барлық жақтар */ .padding-all { padding: 20px; } /* Қысқа жазу - 2 мән (vertical horizontal) */ .padding-vh { padding: 20px 15px; } /* Қысқа жазу - 3 мән (top horizontal bottom) */ .padding-three { padding: 10px 15px 20px; } /* Қысқа жазу - 4 мән (top right bottom left) */ .padding-full { padding: 10px 15px 20px 5px; } /* Пиксель емес өлшемдер */ .padding-em { padding: 1em; } .padding-rem { padding: 1.5rem; } .padding-percent { padding: 5%; } /* Батырмалар үшін padding */ .button-padding { padding: 12px 24px; /* vertical horizontal */ } /* Card стильдері */ .card-padding { padding: 24px; /* Барлық жағына бірдей */ } /* Input элементтері */ .input-padding { padding: 10px 15px; /* Ыңғайлы енгізу */ } /* Asymmetric padding */ .asymmetric-padding { padding-top: 30px; padding-bottom: 15px; padding-left: 20px; padding-right: 20px; }

⚖️ Margin vs Padding

🔴 Margin

  • Орналасуы: Элементтің сыртынан
  • Background: Фон түсіне әсер етпейді
  • Clickable: Басуға келмейді
  • Collapsing: Margin collapse болады
  • Negative мәндер: Қолдауы бар
  • Auto мәні: Қолдауы бар
  • Пайдаланылады: Элементтер арасындағы аралық
Margin мысалы

🔵 Padding

  • Орналасуы: Элементтің ішінен
  • Background: Фон түсін қабылдайды
  • Clickable: Басуға келеді
  • Collapsing: Collapse болмайды
  • Negative мәндер: Қолдауы жоқ
  • Auto мәні: Қолдауы жоқ
  • Пайдаланылады: Контент пен border арасы
Padding мысалы

🤔 Қашан қайсысын қолдану керек?

Margin пайдалану:
  • Элементтер арасындағы аралық
  • Блоктарды центрлеу
  • Layout spacing
  • Responsive breakpoints
Padding пайдалану:
  • Контент пен border арасы
  • Clickable area ұлғайту
  • Батырма, карта ішкі кеңістігі
  • Background area ұлғайту

❌ Negative Margin

Теріс мәндер: Элементтерді жақындату және overlap жасау

/* Negative margin негіздері */ .overlap-top { margin-top: -20px; /* Жоғарыға қарай жылжыту */ } .overlap-left { margin-left: -15px; /* Солға қарай жылжыту */ } /* Элементтерді жақындату */ .tight-spacing .item { margin-bottom: -5px; } /* Overlapping cards */ .card-stack .card { margin-top: -10px; } .card-stack .card:first-child { margin-top: 0; /* Бірінші элементке negative margin жоқ */ } /* Pull element from flow */ .pull-out { margin-left: -30px; margin-right: -30px; } /* Breadcrumb navigation */ .breadcrumb-item { margin-right: -10px; padding-right: 20px; position: relative; } /* Image overlay */ .image-overlay { margin-top: -50px; margin-left: 20px; z-index: 2; position: relative; } /* Responsive negative margins */ .responsive-pull { margin: 0 -15px; } @media (max-width: 768px) { .responsive-pull { margin: 0 -10px; /* Мобилде кішірек pull */ } } /* Negative margin grid */ .negative-grid { margin: -10px; /* Container-ға negative margin */ } .negative-grid .item { padding: 10px; /* Items-ке positive padding */ }
⚠️ Ескерту: Negative margin-ды мұқият қолданыңыз. Responsive дизайнда проблема тудыруы мүмкін.

📉 Margin Collapsing

Margin құлау: Вертикалды margin-дердің біріктірілуі

/* Margin collapsing мысалдары */ /* Стандартты collapsing */ .element-1 { margin-bottom: 20px; } .element-2 { margin-top: 30px; /* Нәтиже: 30px аралық (20px емес 50px) */ } /* Collapsing-ті болдырмау */ .prevent-collapse-1 { margin-bottom: 20px; padding-bottom: 1px; /* Кез келген padding */ } .prevent-collapse-2 { margin-top: 30px; /* Енді 51px аралық болады */ } /* Border арқылы болдырмау */ .border-prevent { border-bottom: 1px solid transparent; margin-bottom: 20px; } /* Flexbox margin collapse болмайды */ .flex-container { display: flex; flex-direction: column; } .flex-item { margin: 20px 0; /* Collapsing жоқ */ } /* Grid-те де collapsing жоқ */ .grid-container { display: grid; gap: 20px; /* Margin орнына gap қолдану */ } /* Inline-block collapsing болдырмайды */ .inline-block-item { display: inline-block; margin: 20px 0; } /* Float элементтер collapsing болмайды */ .float-item { float: left; margin: 20px 0; } /* Position: absolute/fixed collapsing болмайды */ .positioned-item { position: absolute; margin: 20px 0; } /* Overflow: hidden parent collapsing болдырмайды */ .overflow-parent { overflow: hidden; } .overflow-child { margin: 20px 0; /* Parent-пен collapsing болмайды */ }

📱 Responsive Spacing

Бейімделмелі аралық: Әртүрлі экран өлшемдеріне spacing

/* Responsive spacing стратегиялары */ /* Mobile-first подход */ .responsive-spacing { margin: 10px; padding: 15px; } /* Tablet */ @media (min-width: 768px) { .responsive-spacing { margin: 20px; padding: 25px; } } /* Desktop */ @media (min-width: 1024px) { .responsive-spacing { margin: 30px; padding: 40px; } } /* Viewport unit-тер */ .viewport-spacing { margin: 2vw; /* Viewport width-ке байланысты */ padding: 3vh; /* Viewport height-ке байланысты */ } /* Container queries (жаңа) */ @container (min-width: 400px) { .container-responsive { padding: 20px; } } /* Clamp функциясы */ .fluid-spacing { margin: clamp(10px, 5vw, 50px); padding: clamp(15px, 4vw, 60px); } /* Grid gap responsive */ .responsive-grid { display: grid; gap: 10px; } @media (min-width: 768px) { .responsive-grid { gap: 20px; } } @media (min-width: 1024px) { .responsive-grid { gap: 30px; } } /* Flexbox gap responsive */ .responsive-flex { display: flex; gap: 10px; flex-wrap: wrap; } @media (min-width: 768px) { .responsive-flex { gap: 20px; } } /* Component-based spacing */ .btn { padding: 8px 16px; } @media (min-width: 768px) { .btn { padding: 12px 24px; } } .card { padding: 16px; margin-bottom: 16px; } @media (min-width: 768px) { .card { padding: 24px; margin-bottom: 24px; } } /* Typography spacing */ .heading { margin-bottom: 0.5em; } .paragraph { margin-bottom: 1em; } /* Section spacing */ .section { padding: 40px 0; } @media (min-width: 768px) { .section { padding: 60px 0; } } @media (min-width: 1024px) { .section { padding: 80px 0; } }

📐 Responsive spacing принциптері:

  • Mobile-first: Кіші мәндерден бастаңыз
  • Пропорционалдылық: Spacing арақатынасын сақтаңыз
  • Touch targets: Мобилде басуға ыңғайлы етіңіз
  • Viewport units: vw, vh пайдаланыңыз
  • Container queries: Жаңа стандартты пайдаланыңыз

🎯 Spacing дизайн принциптері

✅ Дұрыс тәжірибелер:

  • Spacing scale: Математикалық прогрессия қолданыңыз (4, 8, 16, 24, 32...)
  • Consistency: Бүкіл жобада бірдей spacing жүйесін ұстаныңыз
  • Rhythm: Vertical rhythm сақтаңыз
  • Touch targets: Мобилде минимум 44px басылатын аймақ
  • White space: Кеңістікті дизайн элементі ретінде пайдаланыңыз
  • Semantic spacing: Мағынаға байланысты spacing қолданыңыз

❌ Қате тәжірибелер:

  • Кездейсоқ spacing мәндерін қолдану
  • Барлық жерде бірдей spacing пайдалану
  • Negative margin-ды дұрыс түсінбей қолдану
  • Margin collapsing ескермей layout жасау
  • Responsive spacing ескермеу
  • Gap property-ді білмей ескі әдістерді қолдану

🔢 Ұсынылатын spacing scale:
