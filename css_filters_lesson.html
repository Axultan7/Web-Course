CSS Filter қасиеті туралы

CSS Filter қасиеті HTML элементтеріне (әсіресе суреттерге) визуалды эффекттер қосуға мүмкіндік береді. Photoshop сияқты графикалық редакторлардағы эффекттерді CSS арқылы қолдануға болады.

Filter қасиетінің артықшылықтары:

  • Жылдамдық - браузерде тікелей өңделеді
  • Интерактивтілік - JavaScript арқылы динамикалық өзгертуге болады
  • Кеңістік үнемдеу - бірнеше сурет файлдары керек емес
  • SEO-оңтайландыру - бастапқы сурет сақталады
  • Responsive дизайн - барлық экран өлшемдерінде жұмыс істейді

Filter эффекттерінің мысалдары:

.image { filter: blur(5px) brightness(1.2) contrast(1.5); }

Негізгі CSS Filter функциялары

blur() - Бұлдырлау

Мақсаты: Элементті бұлдырлату

Мәндер: px, rem (0px - бұлдырлау жоқ)

filter: blur(5px);

brightness() - Жарықтық

Мақсаты: Жарықтықты реттеу

Мәндер: сан немесе % (1 = 100% - қалыпты)

filter: brightness(150%);

contrast() - Контраст

Мақсаты: Контрастты өзгерту

Мәндер: сан немесе % (1 = 100% - қалыпты)

filter: contrast(200%);

grayscale() - Сұр түс

Мақсаты: Сұр түске айналдыру

Мәндер: 0-1 немесе 0%-100% (0 - түрлі-түсті, 1 - сұр)

filter: grayscale(100%);

sepia() - Сепия эффектісі

Мақсаты: Ескі фото эффектісі

Мәндер: 0-1 немесе 0%-100% (0 - эффект жоқ, 1 - толық сепия)

filter: sepia(80%);

saturate() - Қанықтық

Мақсаты: Түс қанықтығын өзгерту

Мәндер: сан немесе % (1 = 100% - қалыпты)

filter: saturate(200%);

💡 Маңызды ескертулер:

  • Filter мәндері 0 болса - эффект өшіріледі
  • Brightness пен contrast үшін 1 - қалыпты мән
  • Grayscale пен sepia үшін 0-1 аралығында
  • Бірнеше filter-ді бос орынмен бөліп жазуға болады

Кешенді Filter функциялары

hue-rotate() - Түс айналдыру

Мақсаты: Түс спектрін айналдыру

Мәндер: градуспен (0deg - 360deg)

filter: hue-rotate(90deg);

invert() - Түстерді кері айналдыру

Мақсаты: Негатив эффектісі

Мәндер: 0-1 немесе 0%-100% (0 - кері айналдыру жоқ)

filter: invert(100%);

opacity() - Мөлдірлік

Мақсаты: Элементті мөлдір ету

Мәндер: 0-1 немесе 0%-100% (1 - толық көрінеді)

filter: opacity(50%);

drop-shadow() - Көлеңке

Мақсаты: Көлеңке қосу

Мәндер: x y blur color

filter: drop-shadow(5px 5px 10px #000);

url() - SVG Filter

Мақсаты: SVG filter-дерін қолдану

Мәндер: SVG элементіне сілтеме

filter: url(#my-filter);

Бірнеше filter бірге

Мақсаты: Күрделі эффекттер жасау

Қолданылуы: бос орынмен бөліп жазу

filter: blur(1px) brightness(1.3) saturate(1.5);

Filter функцияларының орындалу реті:

CSS-та filter функциялары солдан оңға қарай орындалады. Мысалы:

filter: brightness(1.5) blur(2px);

Алдымен brightness, содан кейін blur қолданылады. Реттілік маңызды!

Практикалық тапсырмалар

Енді өз білімдеріңізді тексеріп көріңіз. Төмендегі тапсырмаларды орындаңыз:

Тапсырма 1: Винтаж эффектісін жасаңыз

Төмендегі суретке винтаж эффектісін қосыңыз (sepia + contrast + brightness)

.vintage-image { filter: sepia(0%) contrast(100%) brightness(100%); }

Тапсырма 2: Неон эффектісін жасаңыз

Жарқын неон эффектісін жасаңыз (brightness + saturate + hue-rotate)

.neon-image { filter: brightness(100%) saturate(100%) hue-rotate(0deg); }
