CSS Position қасиеті туралы

Position қасиеті элементтің веб-беттегі орналасуын анықтайды. Бұл CSS-тің ең маңызды қасиеттерінің бірі болып табылады.

Position мәндері:

  • static - әдепкі мән (default)
  • relative - қатысты орналастыру
  • absolute - абсолютті орналастыру
  • fixed - тұрақты орналастыру
  • sticky - жабысқақ орналастыру

Static Position

Static - барлық элементтердің әдепкі орналасуы. Элемент құжат ағынында (document flow) қалыпты түрде орналасады.

Top, right, bottom, left қасиеттері static элементтерге әсер етпейді.

.static-element { position: static; /* top, left қасиеттері жұмыс істемейді */ }

Relative Position

Relative - элемент өзінің қалыпты орнына қатысты жылжытылады. Бастапқы орны сақталады (басқа элементтер оның орнын алмайды).

.relative-element { position: relative; top: 20px; left: 30px; }

Absolute Position

Absolute - элемент құжат ағынынан шығарылады және ең жақын positioned parent элементіне (position: static емес) қатысты орналастырылады.

Егер positioned parent болмаса, онда body элементіне қатысты орналасады.

.absolute-element { position: absolute; top: 10px; right: 20px; }

Fixed Position

Fixed - элемент viewport-қа (браузер терезесіне) қатысты тұрақты орналастырылады. Беттің scroll болуына қарамастан бір жерде қалады.

.fixed-element { position: fixed; top: 0; right: 0; }

Fixed мысалы:

Оң жоғарғы бұрыштағы қызыл блокты қараңыз. Ол fixed position қолданады және бет scroll болған кезде де сол жерде қалады.

Sticky Position

Sticky - элемент relative және fixed арасындағы гибридті position. Белгілі бір нүктеге дейін relative, содан кейін fixed ретінде әрекет етеді.

.sticky-element { position: sticky; top: 0; }
