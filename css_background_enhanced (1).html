🌈 Градиенттер (Gradients)

📐 Linear Gradient

Сызықтық градиент - түстер бір бағытта өзгереді

/* Жоғарыдан төменге */ background: linear-gradient(to bottom, #3498db, #2c3e50); /* Сол жақтан оң жаққа */ background: linear-gradient(to right, #e74c3c, #f39c12); /* Бұрышпен */ background: linear-gradient(45deg, #9b59b6, #3498db); /* Пайызбен */ background: linear-gradient(90deg, #e74c3c 0%, #f39c12 50%, #27ae60 100%);

⭕ Radial Gradient

Радиалды градиент - түстер ортадан шетке қарай өзгереді

/* Дөңгелек */ background: radial-gradient(circle, #3498db, #2c3e50); /* Эллипс */ background: radial-gradient(ellipse, #e74c3c, #c0392b); /* Орналасуын көрсету */ background: radial-gradient(circle at top left, #9b59b6, #8e44ad); /* Өлшеммен */ background: radial-gradient(circle 100px, #27ae60, #229954);

🌀 Conic Gradient

Конустық градиент - түстер айналмалы түрде өзгереді

/* Қарапайым conic */ background: conic-gradient(#e74c3c, #f39c12, #27ae60, #3498db, #e74c3c); /* Бұрышпен бастау */ background: conic-gradient(from 90deg, #9b59b6, #3498db, #9b59b6); /* Орталық нүктемен */ background: conic-gradient(at 30% 40%, #e74c3c, #f39c12, #e74c3c);

🖼️ Фон суреті (Background Image)

🏞️ background-image

Элементке фон суретін орнатады

background-image: url('image.jpg'); /* Сурет файлы */ background-image: url('https://...'); /* URL арқылы */ background-image: none; /* Сурет жоқ */ /* Бірнеше сурет */ background-image: url('image1.jpg'), url('image2.jpg');
Фон суреті мысалы
SVG паттерн қолданылған

📐 background-size

Фон суретінің өлшемін басқарады

background-size: auto; /* Бастапқы өлшем */ background-size: cover; /* Контейнерді толық жабу */ background-size: contain; /* Сурет толығымен көрінуі */ background-size: 100px 100px; /* Нақты өлшемдер */ background-size: 50% 50%; /* Пайызбен */

🎯 Практикалық мысалдар

🌅 Hero секциясы

Қош келдіңіз!

Керемет дизайнға арналған фон эффекттері

.hero { background: linear-gradient(rgba(0,0,0,0.5), rgba(0,0,0,0.5)), url('hero-image.jpg'); background-size: cover; background-position: center; color: white; padding: 80px 20px; text-align: center; }

🎨 Анимациялы фон

Анимациялы градиент

Динамикалық фон эффекті

.animated-bg { background: linear-gradient(-45deg, #ee7752, #e73c7e, #23a6d5, #23d5ab); background-size: 400% 400%; animation: gradientShift 4s ease infinite; } @keyframes gradientShift { 0% { background-position: 0% 50%; } 50% { background-position: 100% 50%; } 100% { background-position: 0% 50%; } }

💡 Пайдалы кеңестер

✅ Жақсы практикалар:

  • Мәтін оқылуы үшін жеткілікті контраст қолданыңыз
  • Градиенттер үшін ұқсас түстерді таңдаңыз
  • Фон суреттерінің өлшемін оптимизациялаңыз
  • Мобильді құрылғыларда көрінуін тексеріңіз
  • Background shorthand қасиетін дұрыс қолданыңыз
  • Жүктеу уақытын ескеріңіз

❌ Тыйым салынған:

  • Мәтінді оқуды қиындататын фондар
  • Тым көп түс бір градиентте
  • Үлкен өлшемді оптимизациясыз суреттер
  • Қарама-қарсы түстерді қатар қолдану
  • Фон attachment fixed мобильдерде
  • Тым күрделі паттерндер
